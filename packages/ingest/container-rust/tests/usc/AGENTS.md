# USC Tests Guide

This directory holds United States Code-specific Rust ingest tests.

- Keep adapter, discover, parser, and cross-reference coverage aligned with the USC source modules.
- Update these tests with any USC semantic change.

## Files

- `adapter.rs`: USC adapter tests.
- `cross_references.rs`: USC cross-reference tests.
- `discover.rs`: USC discovery tests.
- `mod.rs`: USC test module exports.
- `parser.rs`: USC parser tests.
