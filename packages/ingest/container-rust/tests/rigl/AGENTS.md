# RIGL Tests Guide

This directory holds Rhode Island-specific Rust ingest tests.

- Keep adapter, discover, and parser coverage aligned with the RIGL source modules.
- Update these tests with any RIGL semantic change.

## Files

- `adapter.rs`: RIGL adapter tests.
- `discover.rs`: RIGL discovery tests.
- `mod.rs`: RIGL test module exports.
- `parser.rs`: RIGL parser tests.
