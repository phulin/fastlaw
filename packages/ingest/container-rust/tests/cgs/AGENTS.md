# CGS Tests Guide

This directory holds Connecticut-specific Rust ingest tests.

- Keep adapter, discover, and parser coverage aligned with the CGS source modules.
- Update these tests with any CGS semantic change.

## Files

- `adapter.rs`: CGS adapter tests.
- `discover.rs`: CGS discovery tests.
- `mod.rs`: CGS test module exports.
- `parser.rs`: CGS parser tests.
