# VT Tests Guide

This directory holds Vermont-specific Rust ingest tests.

- Keep adapter, discover, and parser coverage aligned with the VT source modules.
- Update these tests with any VT semantic change.

## Files

- `adapter.rs`: VT adapter tests.
- `discover.rs`: VT discovery tests.
- `mod.rs`: VT test module exports.
- `parser.rs`: VT parser tests.
