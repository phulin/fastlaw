# MGL Tests Guide

This directory holds Massachusetts-specific Rust ingest tests.

- Keep adapter, discover, parser, and cross-reference coverage aligned with the MGL source modules.
- Update these tests with any MGL semantic change.

## Files

- `adapter.rs`: MGL adapter tests.
- `cross_references.rs`: MGL cross-reference tests.
- `discover.rs`: MGL discovery tests.
- `mod.rs`: MGL test module exports.
- `parser.rs`: MGL parser tests.
