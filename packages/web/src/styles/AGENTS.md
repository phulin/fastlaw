# Styles Guide

This directory holds CSS for the web app and PDF experience.

- Keep styling concerns here.
- Do not encode semantic behavior in CSS naming or structure.
- Preserve the visual separation between the general app and the PDF redline UI.

## Files

- `global.css`: global application styles.
- `pdf-annotations.css`: styles for PDF annotations and amendment snippets.
- `pdf-base.css`: base layout and styling for the PDF app.
- `pdf-instruction-modal.css`: styles for instruction debug or detail modals.
- `pdf-upload.css`: styles for the PDF upload entry flow.
- `pdf-virtualizer.css`: styles for virtualized PDF page rendering.
